# Elevator System Testing (Python + pytest)

This project simulates a simple elevator system and demonstrates basic QA automation using Python and pytest.

## Features
- Elevator movement between floors
- Floor validation (invalid floor handling)
- Weight limit (overload protection)
- Manual simulation script for real-world behavior testing

## Testing
Automated tests are written using pytest and cover:
- Valid floor navigation
- Invalid floor scenarios
- Overload safety logic
- State verification
